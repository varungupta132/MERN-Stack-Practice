# MERN Stack Practice Repository 🚀

This repository is created for **practice and learning purpose**.  
Anyone who is learning **MERN Stack or Backend Development** can use this repo as a reference.

---

## 📌 Purpose of this Repository

- Practice Node.js, Express, and MongoDB
- Understand backend concepts step by step
- Build small projects for learning
- Help beginners revise concepts easily

---

## 📂 What You Will Find Here

This repo contains multiple folders with different practice topics:

- Basic server creation (Node.js)
- GET & POST requests handling
- REST APIs
- File system operations
- MongoDB & Mongoose usage
- Server Side Rendering (EJS)
- Mini projects (To-Do List, URL Shortener, etc.)

---

## 🌐 URL Shortener Project

One of the main projects in this repo.

### Features:
- Convert long URL into short URL
- Redirect short URL to original URL
- Uses MongoDB for storing data
- Built using Express and EJS (SSR)

---

## ⚙️ How to Run

1. Clone the repository:
   git clone https://github.com/varungupta132/MERN-Stack-Practice.git


2. Go to any project folder:

cd project-folder-name


3. Install dependencies:

npm install


4. Run the project:

nodemon app.js
